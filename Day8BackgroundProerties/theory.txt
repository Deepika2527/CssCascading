CSS BACKGROUND PROPERTIES - COMPLETE THEORY NOTES

1. INTRODUCTION TO BACKGROUND PROPERTIES
============================================================

Background properties are used to control the appearance of an
element's background.

Using background properties, we can:

✓ Apply colors
✓ Add images
✓ Control image repetition
✓ Control image position
✓ Control image size
✓ Create gradients
✓ Create parallax effects

Syntax:

selector{
    background-property:value;
}

============================================================
1. BACKGROUND-COLOR
============================================================

Definition:

Used to apply color to an element's background.

Syntax:

background-color: color;

Values:

red
blue
green
yellow
rgb()
rgba()
hex()
hsl()
hsla()
transparent

Examples:

background-color:red;

background-color:#ff0000;

background-color:rgb(255,0,0);

background-color:rgba(255,0,0,0.5);

------------------------------------------------------------
Example
------------------------------------------------------------

div{
    background-color:yellow;
}

============================================================
2. BACKGROUND-IMAGE
============================================================

Definition:

Used to apply an image as the background.

Syntax:

background-image:url("image.jpg");

Example:

div{
    background-image:url("flower.jpg");
}

------------------------------------------------------------
Using Gradient as Background
------------------------------------------------------------

background-image:
linear-gradient(red,blue);

background-image:
radial-gradient(red,blue);

background-image:
conic-gradient(red,blue);

============================================================
3. BACKGROUND-REPEAT
============================================================

Definition:

Controls whether the background image repeats.

Syntax:

background-repeat:value;

Values:

repeat
repeat-x
repeat-y
no-repeat
space
round

------------------------------------------------------------
repeat
------------------------------------------------------------

Default value.

Repeats horizontally and vertically.

background-repeat:repeat;

------------------------------------------------------------
repeat-x
------------------------------------------------------------

Repeats only horizontally.

background-repeat:repeat-x;

------------------------------------------------------------
repeat-y
------------------------------------------------------------

Repeats only vertically.

background-repeat:repeat-y;

------------------------------------------------------------
no-repeat
------------------------------------------------------------

Displays image only once.

background-repeat:no-repeat;

------------------------------------------------------------
space
------------------------------------------------------------

Repeats image with equal spacing.

background-repeat:space;

------------------------------------------------------------
round
------------------------------------------------------------

Repeats and stretches image if necessary.

background-repeat:round;

============================================================
4. BACKGROUND-POSITION
============================================================

Definition:

Controls the starting position of a background image.

Syntax:

background-position:value;

------------------------------------------------------------
Keyword Values
------------------------------------------------------------

left
right
top
bottom
center

Examples:

background-position:left;

background-position:right;

background-position:center;

background-position:top;

background-position:bottom;

------------------------------------------------------------
Combined Values
------------------------------------------------------------

background-position:left top;

background-position:right top;

background-position:left bottom;

background-position:right bottom;

background-position:center center;

------------------------------------------------------------
Percentage Values
------------------------------------------------------------

background-position:50% 50%;

background-position:100% 0%;

background-position:0% 100%;

------------------------------------------------------------
Pixel Values
------------------------------------------------------------

background-position:100px 50px;

Meaning:

100px from left
50px from top

============================================================
5. BACKGROUND-SIZE
============================================================

Definition:

Controls the size of the background image.

Syntax:

background-size:value;

------------------------------------------------------------
Keyword Values
------------------------------------------------------------

auto
cover
contain

------------------------------------------------------------
auto
------------------------------------------------------------

Default value.

Uses original image size.

background-size:auto;

------------------------------------------------------------
cover
------------------------------------------------------------

Image covers the entire container.

May crop some portions.

background-size:cover;

Best for:

Hero Sections
Full Screen Banners

------------------------------------------------------------
contain
------------------------------------------------------------

Entire image becomes visible.

May leave empty spaces.

background-size:contain;

------------------------------------------------------------
Width and Height Values
------------------------------------------------------------

background-size:300px 200px;

background-size:100% 100%;

background-size:50% auto;

============================================================
IMPORTANT DIFFERENCE
============================================================

background-size:cover

✓ Fills entire container
✓ May crop image

------------------------------------------------------------

background-size:contain

✓ Shows full image
✓ May leave empty space

============================================================
6. BACKGROUND-ATTACHMENT
============================================================

Definition:

Determines whether the background image scrolls with content.

Syntax:

background-attachment:value;

Values:

scroll
fixed
local

------------------------------------------------------------
scroll
------------------------------------------------------------

Default value.

Image scrolls with page.

background-attachment:scroll;

------------------------------------------------------------
fixed
------------------------------------------------------------

Image stays fixed.

Creates parallax effect.

background-attachment:fixed;

------------------------------------------------------------
local
------------------------------------------------------------

Background scrolls with element content.

background-attachment:local;

============================================================
7. BACKGROUND-ORIGIN
============================================================

Definition:

Determines where the background image starts.

Syntax:

background-origin:value;

Values:

padding-box
border-box
content-box

------------------------------------------------------------
padding-box
------------------------------------------------------------

Default value.

Starts from padding area.

background-origin:padding-box;

------------------------------------------------------------
border-box
------------------------------------------------------------

Starts from border area.

background-origin:border-box;

------------------------------------------------------------
content-box
------------------------------------------------------------

Starts from content area.

background-origin:content-box;

============================================================
8. BACKGROUND-CLIP
============================================================

Definition:

Determines how far the background extends.

Syntax:

background-clip:value;

Values:

border-box
padding-box
content-box
text

------------------------------------------------------------
border-box
------------------------------------------------------------

Default value.

Background extends under border.

background-clip:border-box;

------------------------------------------------------------
padding-box
------------------------------------------------------------

Stops at padding area.

background-clip:padding-box;

------------------------------------------------------------
content-box
------------------------------------------------------------

Only content area gets background.

background-clip:content-box;

------------------------------------------------------------
text
------------------------------------------------------------

Background appears inside text.

background-clip:text;

Example:

background:linear-gradient(red,blue);
-webkit-background-clip:text;
color:transparent;
